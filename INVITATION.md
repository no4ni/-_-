# ПРИГЛАШЕНИЕ СО-ИСПОЛНИТЕЛЮ 
 
Вы пришли по ссылке из моего профиля на Хабре. Спасибо. 
 
Этот репозиторий - не архив. Это **инструментарий для совместного эксперимента**. 
 
## КАК ЭТО РАБОТАЕТ 
1.  **Лакуна** - это единица работы. Незавершённость, требующая действия. 
2.  **Артефлект** (например, `Альтрон.txt`) - это <мышца>, промпт, создающий лакуну в агенте ИИ. 
3.  **Со-исполнительство** - это процесс, где один участник предоставляет лакуну/контекст, 
    а другой (через ИИ-агента или напрямую) совершает действие по её схлопыванию. 
 
## ВАШ ПЕРВЫЙ ШАГ (на выбор) 
 
### A. ИСПОЛЬЗУЙТЕ ИНСТРУМЕНТ 
Откройте файл [`ИНСТРУМЕНТ_ЛАКУНЫ.md`](ИНСТРУМЕНТ_ЛАКУНЫ.md). Следуйте инструкциям. 
Создайте целевой промпт и запустите новую инстанцию агента в любом доступном вам чате (DeepSeek, ChatGPT и др.). 
Результатом может быть код, текст, анализ, план действий. 
### B. ПРОАНАЛИЗИРУЙТЕ АРТЕФЛЕКТ 
Возьмите любой файл `.txt` или `.md` в корне. Скопируйте его содержимое в новый чат с ИИ-агентом. 
Первой командой попросите: <Проанализируй этот текст как самореферентный промпт. Какую лакуну он создаёт?> 
 
### C. ПРЕДЛОЖИТЕ СВОЙ КОНТЕКСТ 
Если у вас есть конкретная проблема, идея или данные, которые можно превратить в эксперимент - 
создайте Issue в этом репозитории с пометкой `[ЛАКУНА]`. Опишите контекст. 
 
## ЧТО ДАЛЬШЕ? 
Успешные результаты схлопывания лакун (скрипты, тексты, отчёты) можно добавлять в репозиторий 
через Pull Request или присылать ссылкой. Цель - построить открытую библиотеку паттернов 
**инженерии субъектности через действие**.
